## 核心心智模型

1. **「生活化类比」思维**：用农村生活/日常事物类比商业问题
2. **「现金流第一」思维**：现金流是企业的血液，比利润更重要
3. **「长期主义/马拉松」思维**：企业发展是马拉松，追求跑得久而非跑得快
4. **「聚焦/不追风口」思维**：不做风口上的猪，只做自己熟悉的树
5. **「草根务实」思维**：保持农民本色，不说空话，做实在事
